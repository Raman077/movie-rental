# Movie Rental 

## Setup
### 1. Install project dependencies  

```sh
$ npm install
```

### 2. Setup database 
```sh
$ cd docker-database
$ docker-compose up -d
```

### 3. Start the application

```sh
$ npm start

Server is running at http://127.0.0.1:3000
```

## Tests

Run `npm test` from the root folder.
